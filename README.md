## Installation
Install using [npm](http://npmjs.com):
```
npm install @ox2/button --save
```
Install using [yarn](http://yarnpkg.com):
```
yarn add @ox2/button
```

## Usage
```js
import Button from '@ox2/button/Button';

<Button btn="action small raised">Done</Button>

```
